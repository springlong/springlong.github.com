vm，使用avalon.define方法生成，必须带$id属性。

指令，以ms-开头的属性及双花括号的插值表达式。

圈定作用域，使用ms-controller告诉框架，只处理这个范围内的标签。

引导符，使用@来告诉框架这些变量是来自vm的。

<script>
    var vm = avalon.define({
        $id: "test",
        name: "龙泉"
    });
</script>
<div ms-controller="test">
    <input ms-duplex="@name">
    <p>Hello,{{@name}}!</p>
</div>

Hello,{{@name}}!
