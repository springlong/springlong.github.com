认识Vue！

通过new Vue({})创建一个Vue的实例。

通过v-model指令进行表单数据的双向绑定。

通过{{}}这样一对双大括号的“Mustache”语法在HTML代码中插入文本值。

通过v-text指令绑定标签需要输出的文本内容。

通过v-html指令绑定标签需要输出的HTML代码。

<div id="vueTest">
 	<p>请在这里输出内容：<input type="text" size="50" v-model="message" placeholder="Hello World！"></p>
 	<p>“Mustache”语法 (双大括号) 的文本插值：{{message || 'Hello World！'}}</p>
 	<p>通过v-text指令输出普通文本：<span v-text="message"></span></p>
 	<p>通过v-html指令输出HTML代码：<span v-html="message"></span></p>
</div>
<script>
	// Vue实例化
	var vm = new Vue({
		el: '#vueTest',
		data: {
			message: '<b>三种内容输出方式的对比</b>'
		}
	});
</script>
