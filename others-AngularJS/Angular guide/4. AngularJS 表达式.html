AngularJS 表达式

AngularJS 表达式写在双大括号内：{{ expression }}。

AngularJS 表达式把数据绑定到 HTML，这与 ng-bind 指令有异曲同工之妙。

AngularJS 将在表达式书写的位置"输出"数据。

AngularJS 表达式 很像 JavaScript 表达式：它们可以包含文字、运算符和变量。

<div ng-app="">
	<!-- AngularJS 数字 -->
	<hr>
	<div>
 		<p>两数之和： {{ 5 + 5 }}</p>
	</div>
	<div ng-init="quantity=1;cost=5">
		<p>总价： {{ quantity * cost }}</p>
	</div>
	<!-- AngularJS 字符串 -->
	<hr>
	<div>
		<input type="text" ng-model="firstName" ng-init="firstName='Jerry'" placeholder="firstName">
		+
		<input type="text" ng-model="lastName" ng-init="lastName='Sun'" placeholder="lastName">
 		<p>您的名字： {{ firstName + ' ' + lastName }}</p>
	</div>
	<!-- AngularJS 对象 -->
	<hr>
	<div ng-init="person={firstName:'Jerry', lastName:'Sun'}">
		<p>姓啥： {{ person.lastName }}</p>
	</div>		
	<!-- AngularJS 数组 -->
	<hr>
	<div ng-init="points=[1,15,19,2,40]">
		<p>第三个值为 {{ points[2] }}</p>
	</div>
</div>

+

您的名字： {{ firstName + ' ' + lastName }}

姓啥： {{ person.lastName }}

AngularJS 表达式 与 JavaScript 表达式

类似于 JavaScript 表达式，AngularJS 表达式可以包含字母，操作符，变量。

与 JavaScript 表达式不同，AngularJS 表达式可以写在 HTML 中。

与 JavaScript 表达式不同，AngularJS 表达式不支持条件判断，循环及异常。

与 JavaScript 表达式不同，AngularJS 表达式支持过滤器。
